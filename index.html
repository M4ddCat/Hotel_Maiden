Дизайн - кейс приложения для бронирования номера в отелях.

Задача
Данное приложение нужно для облегчения подбора и бронирования номера в отелях.
Целевая аудитория

Данное приложение создавалось, с расчётом на то, что этим приложением будут пользоваться люди от 21 до 55 лет.

Пользовательский сценарий.
При запуске приложения, пользователю предлагается заполнить данные, которые нужны для отбора наиболее подходящих для него номеров в отелях (цена, расположение, дата пребывания).
Если поля заполнены не полностью, пользователю выдаёт ошибку, где его просят их дозаполнить.
После чего пользователю выдаётся перечень подходящих по требованиям номеров в отелях, из которых он выбирает наиболее подходящий для себя номер.
При нажатии на один из предложенных номеров открывается более подробная информация о нём.
Если пользователю всё нравится, то он переходит к оформлению бронирования номера, где ему нужно будет ввести свои личные данные. В противном же случае, если пользователю не нравится номер, он возвращается к выбору номеров.
Если поля заполнены не полностью, или в неправильном формате ( например цифры и символы вместо имени), пользователю выдаёт ошибку, где его просят их дозаполнить или исправить.
После заполнения своих данных пользователь должен подтвердить согласие по обработке личной информации, и при удачной попытке связи сервера с отелем, ему выводится окно, где говорится об успешном бронировании. При неудачной попытке связи с отелем пользователю выводит окно с ошибкой, где пользователя просят повторить запрос позже, и ниже кнопка выхода из приложения.
При успешном завершении данного сценария пользователю выдаётся инструкция, что делать в отеле, чтобы выкупить забронированный номер, если бронь прошла успешна, а так-же ниже кнопка выхода из приложения.

Я выбрал белый цвет для фона, чтобы приложение выглядело светлым, а так-же коричневый цвет как у дерева для элементов, который похож на деревянную мебель в отелях.
На стартовом экране с логотипом используется декоративный шрифт "Solvay" с использованием тёмного цвета, чтобы был контраст с фоном.
В основной части приложения испольуется шрифт Roboto с использованием тёмного цвета на белом фоне, и белого цвета на фоне элементов управления.
